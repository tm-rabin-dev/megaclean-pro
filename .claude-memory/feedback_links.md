---
name: Always give direct links
description: User wants direct clickable URLs, not just domain names to navigate manually
type: feedback
---
Always provide full direct URLs (e.g. https://connectonline.asic.gov.au/...) not just domain names (e.g. "go to asic.gov.au").

**Why:** User finds it faster and easier to click directly rather than navigate manually.

**How to apply:** Any time a website, tool, dashboard, or resource is referenced — give the full direct URL if known with confidence.

Always put anything the user needs to copy (email addresses, phone numbers, ABNs, codes, keys) in fenced code blocks (triple backticks) — NOT inline backticks. Fenced blocks show the copy icon on the far right which is what the user wants. Never write copyable values as plain text.
